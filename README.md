# AI Learning Hub 🚀

2026년 최신 웹 기술과 디자인 트렌드를 반영한 **AI 학습 자료 아카이브 웹앱**입니다. 교육 기관별로 엄선된 AI 학습 가이드를 공유하고, 안전한 비밀번호 시스템을 통해 콘텐츠를 보호합니다.

## ✨ 주요 기능
- **Bento Grid Layout**: 직관적이고 세련된 카드형 인터페이스.
- **Access Control**: 관리자가 발급한 6자리 비밀번호를 통한 자료 접근 제한.
- **Admin Dashboard**: 기관 및 자료 등록, 비밀번호 자동 생성 기능을 담은 관리자 전용 페이지.
- **Real-time Updates**: Firebase Firestore를 통한 실시간 데이터 동기화.
- **Responsive Design**: 모바일과 데스크톱 모두에 최적화된 인터페이스.

## 🛠 기술 스택
- **Framework**: Next.js 15+ (App Router)
- **Database/Auth**: Firebase (Firestore, Auth)
- **Styling**: Tailwind CSS V4, Motion (Framer Motion)
- **Icons**: Lucide-React

## 🔗 관련 링크
- **제작**: [아크랩스 홈페이지](https://litt.ly/aklabs)

## 📝 관리자 가이드
1. `/login` 페이지에서 관리자 계정으로 로그인합니다. (설정된 관리자 이메일: `mosebb@gmail.com`)
2. `/admin` 대시보드에서 교육 기관을 먼저 등록합니다.
3. 등록된 기관을 선택하여 자료 제목, 설명, 가이드 URL, 썸네일 URL을 입력합니다.
4. 비밀번호 생성 아이콘을 눌러 6자리 랜덤 비밀번호를 발급한 뒤 저장합니다.
